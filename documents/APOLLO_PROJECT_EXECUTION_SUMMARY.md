## Data Enrichment & Validation Standards

### Email Verification Standards

**Verification Hierarchy:**
1. **Apollo VERIFIED_CURRENT** (preferred): Email verified within 90 days = HIGHEST confidence
2. **Apollo VERIFIED**: Email verified but not current = HIGH confidence
3. **Secondary Source Verified**: Institution directory, LinkedIn = MEDIUM confidence
4. **Inferred Pattern**: Email pattern matched to institution format = LOW confidence
5. **UNVERIFIED**: No verification sources = EXCLUDE

**Target Verification Rate:** 95%+ emails with VERIFIED or VERIFIED_CURRENT status

### Phone Number Standards

**Source Priority:**
1. Apollo phone field (primary)
2. Institution directory (department or main line)
3. LinkedIn profile (if visible)
4. Institutional website (department staff page)

**Acceptance Criteria:**
- Format validation: (XXX) XXX-XXXX (US) or +country-number (international)
- Reasonableness check: Matches institution phone range
- No test/placeholder numbers (1234567890)

**Fallback:** If no phone available but email VERIFIED, this is acceptable

### LinkedIn Profile Standards

**Validation Checklist:**
- [ ] Profile exists and is publicly accessible
- [ ] Current job title matches Apollo title (exactly or similar)
- [ ] Current company matches Apollo institution
- [ ] Employment start date is 1-5 years ago (reasonable tenure)
- [ ] No job change indicators (recently departed institution)
- [ ] LinkedIn activity recent (update within 6 months)

**Target Confirmation Rate:** 90%+ LinkedIn profile match

### Institution Fit Standards

**Alumni Base Validation:**
- Minimum: 500+ alumni (beachhead requirement)
- Estimation: Organization employee count × institution-type multiplier (2-5x)
- Cross-reference: Official alumni data from institution website when available

**Institution Type Validation:**
- Research Universities: R1/R2 Carnegie Classification + IPEDS verification
- Liberal Arts: Baccalaureate classification + selective enrollment verification
- Community Colleges: Public institution + associate degree verification
- Professional Schools: Specialized accreditation (ABA, LCME, AACSB)

### Decision-Maker Authority Standards

**Qualified Roles:**
- VP-level: Full authority for strategy and budget
- Director-level: Significant authority for operations and outcomes
- Dean-level: Full authority in academic domain
- Chief Officer (CIO, CDO, CAO): Full authority in specialized domain

**Minimum Authority Level:** Director or equivalent (must have decision-making power)

**Disqualified Roles:**
- Faculty (unless administrative leadership)
- Individual contributor/analyst level
- Support/administrative roles without management authority

## Quality Assurance & Validation Process

### Pre-Export Quality Checklist

**Completeness:**
- [ ] All 50+ records present
- [ ] All 9 columns populated (Institution Name, Contact Name, Title, Email, Phone, LinkedIn, Size, Alumni Base, Research Intensity)
- [ ] No blank required fields (Name, Email, Title mandatory)

**Accuracy:**
- [ ] Email verification rate ≥95%
- [ ] LinkedIn confirmation rate ≥90%
- [ ] 10% sample (5-6 records) manually verified against sources
- [ ] Zero invalid email addresses (no test formats)

**Format:**
- [ ] CSV UTF-8 encoding
- [ ] Phone numbers: (XXX) XXX-XXXX format
- [ ] Emails: lowercase, no spaces
- [ ] URLs: HTTPS format, no query parameters
- [ ] Alumni base: Numeric with "+" (e.g., "10000+")
- [ ] Research Intensity: Only values "High", "Medium", "Low"

**Deduplication:**
- [ ] Zero duplicate email addresses
- [ ] Zero duplicate LinkedIn URLs
- [ ] Zero duplicate institutional records (except where multiple qualified contacts)
- [ ] Cross-institution duplicates reviewed and consolidated

**Quality Scoring:**
- [ ] Average quality score: 85+/100
- [ ] Minimum individual score: 70/100
- [ ] Records below 60/100: Documented or excluded

### Final Sign-Off

**Quality Gate Sign-Off:**
- [ ] Data meets all quality thresholds
- [ ] All validation checks completed
- [ ] No data integrity issues identified
- [ ] Ready for export and sales team handoff

**Export Sign-Off:**
- [ ] CSV file opens in Excel without errors
- [ ] Column headers match specification
- [ ] No encoding errors or special character issues
- [ ] File size and record count reasonable (50+ records = 15-30 KB)
