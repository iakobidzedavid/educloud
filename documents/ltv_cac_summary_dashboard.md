- Renewal rate: 85% (industry standard)
- Expansion: 5% annual ACV increase (add-on modules)
- Gross margin: 82%
- Discount rate: 10% (NPV calculation)

**Calculation:**
```
Year 1: $22,000
Year 2: $19,800 × 1.05 = $20,790 (85% retention, 5% expansion)
Year 3: $17,672 × 1.05 = $18,556
Year 4: $15,821 × 1.05 = $16,612
Year 5: $14,119 × 1.05 = $14,825

Total revenue (undiscounted): $93,783
Less COGS (18%):               -$16,881

(Conservative LTV figure)
```

**Key insight:** LTV remains strong even accounting for realistic churn and discounting for time value of money.

## 5. Sensitivity Analysis: Key Variables

### Impact of ACV Variation

```
If ACV = $16,000:     LTV:CAC = 26.2:1  | Payback = 1.2 mo | ✓ Still viable
If ACV = $22,000:     LTV:CAC = 46.4:1  | Payback = 1.2 mo | ✓ Base case
If ACV = $28,000:     LTV:CAC = 52.2:1  | Payback = 1.1 mo | ✓ Stretch goal
```
**Conclusion:** LTV:CAC remains healthy across $16K-$28K ACV range.

### Impact of Churn/Retention

```
If Retention = 75%:   LTV = $67,500  | LTV:CAC = 38.6:1 | Payback still strong
If Retention = 85%:   LTV = $81,180  | LTV:CAC = 46.4:1 | Base case
If Retention = 95%:   LTV = $95,000  | LTV:CAC = 54.3:1 | Optimistic
```
**Conclusion:** Model has safety margin. Even 75% retention remains excellent.

### Impact of CAC Increase

```
If CAC = $1,000:      LTV:CAC = 81.2:1 | Payback = 0.8 mo | Optimistic
If CAC = $1,750:      LTV:CAC = 46.4:1 | Payback = 1.2 mo | Base case
If CAC = $2,625:      LTV:CAC = 30.9:1 | Payback = 1.75 mo | Still strong
If CAC = $5,000:      LTV:CAC = 16.2:1 | Payback = 3.3 mo | Marginal
```
**Conclusion:** Model can absorb up to 50% CAC overrun and remain strong.

### Impact of Gross Margin Erosion (due to higher support)

```
If COGS = 15% (lower support): Margin = 85% | LTV = $88,400 | LTV:CAC = 50.5:1 ✓
If COGS = 18% (baseline):      Margin = 82% | LTV = $81,180 | LTV:CAC = 46.4:1 ✓
If COGS = 25% (higher support):Margin = 75% | LTV = $71,000 | LTV:CAC = 40.6:1 ✓
If COGS = 35% (very high):     Margin = 65% | LTV = $52,700 | LTV:CAC = 30.1:1 ✓
```
**Conclusion:** Even with 35% COGS (poor outcome), LTV:CAC remains >30:1.
