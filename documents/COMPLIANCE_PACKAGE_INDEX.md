## FAQ: Common Questions About This Package

**Q: Is this package sufficient for pilot launch?**  
A: Yes, it provides complete FERPA compliance framework. You still need to:
- Execute institutional DUAs (external counsel)
- Implement technical controls (engineering)
- Obtain cyber insurance (insurance broker)
- Create incident response procedures (legal + security)

**Q: How much will compliance cost?**  
A: Estimated $20-35K:
- External legal counsel: $8-18K
- Cyber insurance: $3-8K annually
- Engineering time: ~$5-10K (if paying $150/hr for 30-50 engineering hours)

**Q: Can we do this faster than 6-8 weeks?**  
A: Unlikely. Critical path is institutional legal review (2-3 weeks uncontrollable). Technical work can parallelize, reducing total time to ~5-6 weeks if aggressive.

**Q: Do we need external legal counsel?**  
A: Recommended for FERPA expertise + credibility with institutions. Internal counsel can handle simpler items; external counsel (higher ed law firm) for FERPA framework + DUA negotiation.

**Q: What if we don't implement all controls before pilot launch?**  
A: High risk. Strongly recommend completing Phase 1 checklist entirely. Do NOT proceed with institutional data access without:
- Institutional DUAs signed
- Database encryption enabled
- Role-based access control implemented
- Audit logging operational

**Q: Who should be the Privacy Lead?**  
A: Ideally someone with:
- FERPA/higher ed compliance experience (preferred but not required)
- Project management skills
- Legal/regulatory background
- Ability to interface with institutions and counsel

**Q: Can we use a vendor for compliance?**  
A: Partially. Some vendors provide:
- Compliance consulting (helps, but you still need internal owner)
- Compliance tools (helps with logging/reporting, but you implement controls)
- Insurance/risk management (outsourceable to broker)
